CmUtils::RenumEx

Renumbers a pair of .exnode and .exelem files with specified node and element offsets.

renumEx

Takes the following arguments:

nodefile
nodeoffset
elemfile
elemoffset
faceoffset
lineoffset
verbose

Only the last argument is optional. Arguments may either be passed as a list, or as a hash array with the above names as the hash keys.
